# Check Status

```bash
https://staging.paysecure.net/api/v1/purchases/{Session ID}/
```

This endpoint retrieves the details of a purchase identified by the provided session ID.

`Note that this API only respond once the SessionURL, received in the reposne for session API, is clicked.`

**AUTHORIZATION** Bearer Token

___

__Token__ &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; `{{API Key}}`

### HEADERS

___

__Authorization__  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;   `{{API Key}}`
